## About Me

<img align="right" width="250" src="https://i.pinimg.com/originals/45/b1/0c/45b10c9f93d42eafee239bfa7e7075fe.gif" />

Final-year **Software Engineering** student from **Jakarta, Indonesia**, focused on building clean, scalable backend systems and mobile applications.

My current stack: building **APIs** with Laravel & Go, crafting mobile experiences with **Flutter**, and managing infrastructure with Docker & Linux.

I'm the type who reads docs for fun and always has too many side projects running. I believe good code is not just about making things work — it's about making them *easy to live with*.
